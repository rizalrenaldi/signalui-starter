## Get Started

```
# Clone repo
git clone https://github.com/rizalrenaldi/signalui-starter.git
cd signalui-starter

# Install dependencies
npm i

# Run dev server - available at http://localhost:1234
npm run dev

# Run production build
npm run build
```
